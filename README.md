# Ahorcado
_Trabajo final para el curso intensivo del centro de formación de CILSA, orientado a aprender conceptos sobre programación en JavaScript, tales como el uso de variables, operadores, arreglos, estructuras de control, funciones, accesibilidad, validación de datos y manipulación del DOM._
## Sobre el proyecto
Se trata de una versión web interactiva del clásico juego del Ahorcado, enfocada en adivinar palabras.
## Características
**Interacción:** El usuario puede jugar utilizando clics en el teclado virtual en pantalla o tipeando directamente con su teclado.
**Control de Estados:** Las letras se deshabilitan automáticamente una vez utilizadas para evitar ingresos duplicados. 
**Actualización:** La imagen del ahorcado cambia en base a la cantidad de intentos fallidos restantes (en este caso, con un máximo de 5 errores).  
## Cómo jugar 🎮
1. Lee la pista que aparece en la parte superior de la pantalla.
2. Ingresa una letra usando el teclado virtual en pantalla o tu teclado.
3. Si aciertas, la palabra completa se revelará en verde.
4. Si te equivocas, la letra se marcará como error, se dibujará una parte del ahorcado y perderás un intento.
5. Haz clic en "Cambiar palabra" en cualquier momento para reiniciar el tablero con un nuevo desafío.
